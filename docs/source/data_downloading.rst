Data downloading module
========================

| The data downloading module within the EarthquakeForecast package offers a versatile and user-centric solution for acquiring earthquake-related data. This module enables users to effortlessly gather seismic information based on defined time intervals and geographical parameters. By interfacing with reputable data centers, such as USGS, ISC, and IRIS, this module ensures the retrieval of accurate and up-to-date data, serving as a foundational step in earthquake analysis and prediction.
